📄 Gestión de Folletos

Subí los PDFs de cada campaña y configurá el año del sistema

🗓️ Año de campaña

El año se usa para construir el nombre del PDF (ej: folleto_c626.pdf = campaña 6, año 26). Cambialo cuando empiece el nuevo año sin tocar código.

ℹ️ Este valor se guarda en el sistema y el panel de clientas lo usa automáticamente. Cuando sea 2027 → ponés 27.

📤 Subir folleto al repositorio

Subí el PDF de cada campaña directamente desde acá. Se sube al repo de GitHub automáticamente. Necesitás un GitHub Personal Access Token con permiso contents:write.

¿Cómo obtengo el token? ▼
1. Andá a github.com → tu perfil → Settings
2. Al final del menú izquierdo: Developer settings
3. Personal access tokens → Tokens (classic) → Generate new token
4. En scopes: marcá solo ✅ repo (o public_repo si el repo es público)
5. Copiá el token y pegalo arriba

⚠️ El token se guarda solo en este navegador (localStorage). Nadie más lo ve.
